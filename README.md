# MiaEntity Add-ons

Repository di add-on personalizzati per Home Assistant.

## Add-on disponibili

### [HistoLite](histolite/)
Gestione intelligente della history dei sensori Home Assistant.  
Alleggerisce il database SQLite con 4 strategie di purge e appiattimento dei dati storici.

## Installazione

1. In Home Assistant vai su **Impostazioni → Add-on → Store → ⋮ → Repository**
2. Aggiungi: `https://github.com/abiale85/ha-addons`
3. Cerca e installa gli add-on disponibili
